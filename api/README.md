
# tut-jobopas REST API
Documentation for tut-jobopas API. Demo is found at http://tut-jobopas.herokuapp.com/api/.

### `/api/tag` - Tags for courses

#### `GET /api/tag`
List all tags.

#### `POST /api/tag`
Create a new tag.

#### `GET /api/tag/:id`
Get single tag.

#### `PUT /api/tag/:id`
Update tag's details.

#### `DELETE /api/tag/:id`
Delete a tag.
